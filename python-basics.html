Basics of Python


Day 1 - Introduction to Python Basics
Overview of Python and its significance in programming. Setting up the Python development environment. Introduction to Python syntax: variables, data types, and basic operations.


Day 2 - Data Types and Control Structures
Recap of Day 1 and Python basics. Exploring more complex data types: lists, tuples, and dictionaries. Introduction to control structures: conditional statements and loops.


Day 3 - Functions and Modules
Introduction to functions: defining, calling, and passing parameters. Organizing code with modules: importing and using modules in Python programs.


Day 4 - Advanced Topics
More advanced Python topics: file handling, exception handling, and libraries. Writing Python scripts to interact with files and handle exceptions. Introduction to Python libraries.


Day 5 - Simple Project Showcase
Final preparations for the project showcase during next mobility. Practice presentations, demonstrations, assignments. Students present their coding projects and demonstrate basic solutions.
